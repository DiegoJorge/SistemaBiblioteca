Introducion al proyecto

El sistema de gestión de bibliotecas de nuestro equipo está hecho con la intención de facilitar el ingreso y registro de empleados y libros dentro de una biblioteca.
Nuestro sistema funciona con una base de datos específica y tiene una interfaz simple y sencilla de comprender y utilizar para facilitar en la mayor parte su uso.

Informacion sobre el proyecto

Nuestro proyecto nació con la idea de ayudar a un sector poco apoyado en ocasiones por la informática brindando opciones claras, sencillas e intuitivas de llevar un registro limpio de los empleados, libros y registros de una biblioteca procurando siempre que sea de un fácil acceso.
Nuestro sistema usa un motor de Windows Forms en Visual Studio 2017 con el cual se han logrado elaborar interfaces sencillas y claras para cada acción que el usuario desee realizar en el momento, utilizando la herramienta de PHP MyAdmin se ha logrado elaborar una base de datos que almacene de una manera eficiente los datos que se utilizarán para el sistema.

Requerimientos minimos del sistema
  • Unidad central de procesamiento: 1GHz.
  • Espacio de memoria RAM: 2GB.
  • Sistema Operativo: Windows 7.
  • Espacio libre: 3GB.


Caracteristicas unicas del programa

El sistema cuenta con las características de cualquier sistema informático actual, un primer uso del sistema y mantenimientos para distintas partes de la base de datos sobre la cual se trabajó el sistema, posee una base de colores sólida y amigable por la combinación suave de azules y celestes.
Cuenta con un sistema de conexión por medio de PhPMyAdmin a una base de datos de donde se extraen la información para que los mantenimientos funcionen correctamente.


Preguntas Frequentes

  • Que tanto se tarda en descargarse el programa?
    • El tiempo de descarga varia de donde se encuentre y de la velocidad de su conexion a internet por lo que se recomienda solo usar conexiones de banda ancha con alta velocidad para descargar de manera optima.
  • Que ocurre despues de que termine la descarga?
    • Una vez completada la descarga, vaya a la ubicación en la que guardó el archivo y haga doble clic en el nuevo icono para iniciar la instalación.
  • Y si la descarga se detiene o interumpe antes de terminar?
    • Si se pierde la conexión mientras se descargan los archivos a través de su explorador web, vuelva a conectarse a Internet e intente realizar la descarga de nuevo.
  • Mi maquina no usa windows, pero quiero usar el programa en ella. Donde encuentro la descarga?
    • Se puede descargar una copia del programa en formato ISO que le permitiria usar el programa. Temporalmente esto no esta disponible.
  • Como puedo revisar si mi maquina necesita la version de 32-bits o la de 64-bits?
    • Haga clic en Inicio, escriba sistema en el cuadro de búsqueda y, a continuación, haga clic en sistema, en la lista del Panel de Control.
    El sistema operativo se muestra como sigue:
    Para un sistema operativo de 64 bits: aparece el sistema operativo de 64 bits del tipo de sistema en sistema.
    Para un sistema operativo de versión de 32 bits: sistemas operativos de 32 bits aparece en el tipo de sistema en sistema.
