Introduction to the project

Our team's administrative system for libraries is made with the intent to make easier adding and keeping a register of workers and books.
the system works with a custom database which has a simple yet easy to understand layout while also making it easy to use on most cases.

Information about our project

Our project was born from the ideal of helping an usually poorly supported community by offering clear, intuitive and intuitive options to keep clean records of employees, books and miscellaneous records while prioritizing for it to be easily accessible.
the system uses a Microsoft Forms engine in Visual studio which we've used to create a simple layout for any action the users may require, using PHP MyAdmin it's also been possible to create a database that efficiently stores the date for the system

Minimum system requirements
  • Central processing unit: 1GHz.
  • RAM: 2GB.
  • Operating system: Windows 7.
  • Free space:3GB


Unique features

The system has the characteristics of any current computer system, a first use of the system and maintenance for different parts of the database on which the system was worked, it has a solid and friendly color base due to the soft combination of blues and light blues
It has a connection system through PhPMyAdmin to a database from where the information is extracted so that the maintenance works correctly.


Frequently asked questions

  • How long does it take to download the program?
    • Download times vary by location and internet connection speed so it is recommended only high-speed broadband connections are to be used to download your file
  • What happens after the dowload is over?
    • After the download has completed, go to the location that you saved the file at and double click on the new icon to start the installation.
  • What if the download stops or gets interrupted before it finishes?
    • If you become disconnected while files are being downloaded through your web browser, reconnect to the internet and retry your download.
  • I have a non-Windows device, yet I want to run the program on it. Where can I find the download for it?
    • You can download an ISO file copy of our program to use with a non-Windows device. Currently this is unavailible as a service.
  • How can I check if i need the 32-bits or the 64-bits version?
    • Click Start, type system in the search box, and then click System in the Control Panel list.
    The operating system is displayed as follows:
    For a 64-bit version operating system: 64-bit Operating System appears for the System type under System.
    For a 32-bit version operating system: 32-bit Operating System appears for the System type under System.
